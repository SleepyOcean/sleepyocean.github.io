# 🎊 iterm配置代理

可以在 ~/.zshrc 或者 ~/.bash_profile 中添加这样的alias：

alias goproxy='export http_proxy=http://127.0.0.1:8087 https_proxy=http://127.0.0.1:8087'
alias disproxy='unset http_proxy https_proxy'
1
2

后续通过goproxy / disproxy进行切换

# ㊙️ Mac 命令

export的用法

export TEST="abc"
1

export作用 在 shell 中执行程序时，shell 会提供一组环境变量。export 可新增，修改或删除环境变量，供后续执行的程序使用。export 的效力仅限于该次登陆操作。

下面看一些例子

// 赋值在test.sh中
export TEST="abc"
source test.sh
echo $TEST //"abc"

sh test.sh
echo $TEST //打印的是空值
