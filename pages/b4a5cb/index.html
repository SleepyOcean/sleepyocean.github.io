Android刷机指南

# 一、常规刷机步骤

# 1、刷入第三方Recovery - TWRP

步骤一、解除BL锁

  1. 打开「开发者模式」。 进入设置，点击我的设备 -> 点击全部参数 -> 一直点版本号，直到出现处于开发者模式
  2. 打开「USB调试」和「OEM解锁」。进入「设置 - 更多设置 - 开发者模式」。OEM解锁必须打开不然没办法刷入第三方Recovery
  3. 下载TWRP工具包。这里我下载的是
  4. 刷入第三方Recovery
# 重启进入 FastBoot 模式
> adb reboot bootloader

# 解锁 flash （这一步一定要确保OEM解锁打开）
> fastboot flashing unlock

# 检查是否解锁成功。出现以下输出表示BL解锁成功
> fastboot oem device-info
(bootloader) Verity mode: true
(bootloader) Device unlocked: true
(bootloader) Device critical unlocked: false
(bootloader) Charger screen enabled: false
(bootloader) Console enabled: false

# 开始刷入twrp
> fastboot flash recovery twrp.img

# 重启设备并进入Recovery
> fastboot reboot recovery

步骤二、TWRP卡刷

  1. 刷入新系统时需要四清。（一定要四清，不然会变砖！！！）
  2. 进入 Install 选项，找到需要安装的卡刷包，安装即可

# 二、MIPAD1刷机指南

1）刷入TWRP

# 进入FastBoot，刷入twrp
> fastboot flash recovery twrp-3.4.0-0-mocha.img
1
2

2）四清

进入TWRP，点击Wipe -> Advance Wipe，选中前四个，滑动开始清理旧系统

3）刷入分区包

四清完成后，回到首页，点击install，选择前面已拷贝的文件 mocha_repartition_1_2GB_RahulTheVirus.zip，并滑动刷入，刷入完成后，会自动重启

4）刷入tos.img

重启进入FastBoot，刷入tos.img

# 进入FastBoot，刷入twrp
> fastboot.exe flash tos tos.img
1
2
  1. 刷入LineageOS系统或其他系统

重启后，进入TWRP，开始安装系统，点击Install，并选中文件 lineage-16.0-20200419-UNOFFICIAL-mocha.zip，滑动开始刷机

# 三、ONEPLUS3T刷机指南

1）使用“一加刷机工具箱”解锁

2）刷入TWRP

# 进入FastBoot，刷入twrp
> fastboot flash recovery twrp-3.4.0-0-mocha.img
1
2

3）双清

进入TWRP，点击“清除（Wipe）”，滑动开始清理

4）使用ADB sideload线刷

进入TWRP，点击“高级（Advance）”，进入后点击“ADB sideload”，进入后滑动按钮开始进入ADB sideload。

在PC上打开控制台（Terminal或CMD）

# 刷入Pixel Experience系统
> adb sideload PixelExperience_Plus_oneplus3-11.0-20210516-1636-OFFICIAL.zip

# 刷入Magisk工具
> adb sideload Magisk-v23.0.zip

# 解决方案

1）刷入PixelExperience一直卡在Logo

先四清，然后输入系统，最后再格式化data

2）“时钟无法同步”以及“WIFI网络连接受限”

# 修改连接地址
> adb shell  "settings put global captive_portal_http_url http://connect.rom.miui.com/generate_204"
> adb shell "settings put global captive_portal_https_url https://connect.rom.miui.com/generate_204"

# 时间同步
> adb shell settings put global ntp_server asia.pool.ntp.org
